# CP8 Client Onboarding Wizard

CP8 adds the client-facing onboarding wizard for Prime2 Client Command OS.

Purpose:
- document who the client is
- capture strategy, market, and pipeline setup
- document lead sources, buyer-list posture, team ownership, and compliance basics
- measure readiness for controlled/manual Prime2 operation
- surface blockers, tasks, and next steps

CP8 is non-live:
- no messages are sent
- no campaigns start
- no provider calls occur
- no DNC provider checks occur
- no 10DLC live registration occurs
- no billing or payment actions occur
- no contract or e-signature actions occur

Core records:
- client business profile
- client strategy profile
- market setup
- pipeline setup and stage templates
- lead-source setup
- buyer-list setup
- team setup checklist
- compliance setup checklist
- first-leads checklist
- onboarding tasks
- onboarding timeline events
- onboarding report

Client-safe rule:
CP8 only creates setup, readiness, and review records. It does not unlock live communication or provider execution.
